# CIS and Networking Projects

This section highlights selected CIS and networking-related coursework.

## Packet Tracer Networking Lab

**Background:** This project involved building and testing a small network using Cisco Packet Tracer.

**Approach:** I configured network devices, assigned IP addresses, tested connectivity, and documented the results.

**Results:** The lab demonstrated basic router and switch configuration, troubleshooting, and network validation.

**Next Steps:** I would improve the project by adding cleaner diagrams, labeled screenshots, configuration notes, and troubleshooting explanations.

## Subnetting Practice

**Background:** Subnetting is a core networking skill used to divide networks into smaller, organized address ranges.

**Approach:** I practiced calculating network addresses, subnet masks, usable host ranges, and broadcast addresses.

**Results:** This work strengthened my understanding of IP addressing and network planning.

**Next Steps:** I would continue practicing subnetting with larger and more complex network designs.
